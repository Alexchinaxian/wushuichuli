# 状态颜色修改计划

## 目标
修改工业控制上位机界面中运行和停止状态的颜色显示。

## 当前颜色分析
通过代码分析，发现状态颜色在以下文件中定义：

1. **StatusToBrushConverter.cs** - 状态指示灯的颜色转换器
   - 运行态：蓝色 #0077FF (RGB: 0, 119, 255)
   - 停止态：浅灰色 #8E8E93 (RGB: 142, 142, 147)

2. **EquipmentItem.cs** - 设备项状态颜色属性
   - 运行态：LimeGreen (系统画笔)
   - 停止态：LightGray (系统画笔)

3. **ProcessUnitModel.cs** - 处理单元边框和背景颜色
   - 多个颜色常量定义（PrimaryColor, OfflineBorderColor等）
   - CurrentBorderColor逻辑使用PrimaryColor表示运行态

## 修改方案
根据用户要求，将颜色修改为：
- **运行态**：绿色 #00FF00 (RGB: 0, 255, 0)
- **停止态**：灰色 #808080 (RGB: 128, 128, 128)

## 风险评估
- **低风险**：颜色修改不影响业务逻辑，只改变视觉效果
- **兼容性**：新颜色可能与现有设计风格冲突，但用户已明确要求
