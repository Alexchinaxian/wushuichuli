## 修改需求
从中间水池的下边中心点垂直向下找一个合适的位置，使用一条竖线和三条横线连接到三个回用泵。

## 修改内容

### 1. 原始设计分析
**原始设计**（修改前）:
```
中间水池下点(1225,520) → 垂直向下到(1225,540)
共享水平线: (1225,540) → (1280,540)  长度55像素
从水平线终点连接到各回用泵:
1#: (1280,540) → (1250,540) → (1250,590)
2#: (1280,540) → (1250,540) → (1250,670)
3#: (1280,540) → (1250,540) → (1250,750)
```
- 问题: 设计复杂，有额外的水平延伸和折返
- 问题: 不是简洁的"一条竖线+三条横线"

### 2. 简洁设计
**简洁设计**（修改后）:
```
一条竖线:
中间水池下边中心点(1225,520) → 垂直向下到(1225,600)  长度80像素

三条横线:
1#回用泵: (1225,600) → 水平向右到(1250,600) → 垂直向上到(1250,590)
2#回用泵: (1225,600) → 水平向右到(1250,600) → 垂直向下到(1250,670)
3#回用泵: (1225,600) → 水平向右到(1250,600) → 垂直向下到(1250,750)
```
- 特点: 一条竖线下来，三条横线出去
- 特点: 设计简洁明了
- 特点: 符合"竖线+横线"的要求
