```

```

### 连接点位置对比
```
原始下点(中心点): X=1225, Y=520

移动距离: 向右25像素
```

### 连线路径变化
**原始路径**:
```
竖线: (1225,520) → (1225,600)  长度80像素
横线: (1225,600) → (1250,600)  长度25像素
```

**移动后路径**:
```
竖线: (1250,520) → (1250,600)  长度80像素
横线: (1250,600) → (1250,600)  长度0像素（同一点）
实际横线: (1250,600) → (1250,600) → 但需要连接到回用泵(1250,Y)
```

实际上，由于连接点向右移动到了X=1250，而回用泵的右边点也是X=1250，所以横线长度变为0。竖线终点和回用泵连接点在同一X坐标上。

这意味着：
- 竖线: `(1250,520) → (1250,600)` 长度80像素
- 从`(1250,600)`直接垂直连接到回用泵，不需要水平移动

## 设计原则

### 1. 连接点优化
- **向右移动**: 从中心点向右移动25像素
- **对齐优化**: 与回用泵右边点X坐标对齐（都是1250）
- **简化连线**: 减少水平线段，连线更加简洁

### 2. 连线简化
- **无水平线**: 连接点与回用泵X坐标对齐，无需水平移动
- **直接垂直**: 从连接点直接垂直连接到回用泵
- **更加简洁**: 连线路径更加简单明了

### 3. 视觉改进
- **整齐对齐**: 连接点与回用泵垂直对齐
- **减少交叉**: 简化连线减少视觉杂乱
- **更加专业**: 对齐设计更加专业规范
