## 修改内容

### 1. 原始连线分析
**原始连线**（修改前）:
```
1#反洗泵左点(900,230) → 直接连接到MBR膜池右边点(875,420)
2#反洗泵左点(900,310) → 直接连接到MBR膜池右边点(875,420)
```
- 路径: 直接斜线连接
- 问题: 两个反洗泵的连线独立，没有"一起竖着"

### 2. 修改后连线
**新连线**（修改后）:
```
1#反洗泵:
(900,230) → 水平向左20像素到(880,230) → 垂直向上40像素到(880,270) → 垂直向下150像素到(880,420) → 水平向左5像素到MBR膜池右边点(875,420)

2#反洗泵:
(900,310) → 水平向左20像素到(880,310) → 垂直向下40像素到(880,270) → 垂直向下150像素到(880,420) → 水平向左5像素到MBR膜池右边点(875,420)
```

## 设计原则

### 1. 连线模式
- **先横**: 两个反洗泵先水平移动到同一垂直线X=880
- **再一起竖**: 从中间点Y=270一起垂直向下到MBR
- **最后横**: 水平移动到MBR连接点

### 2. 汇聚点设计
- **水平汇聚线**: X=880
- **垂直汇聚点**: Y=270（两个反洗泵的中间点）
- **共享垂直线**: 从(880,270)到(880,420)

### 3. 视觉优化
- **整齐统一**: 两个反洗泵的连线模式相同
- **汇聚清晰**: 在中间点汇聚后共享垂直线
- **避免交叉**: 水平移动后垂直向下，路径清晰
