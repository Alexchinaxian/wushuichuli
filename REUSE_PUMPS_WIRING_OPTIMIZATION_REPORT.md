## 优化需求
优化中间水池到三个回用泵，再到用水单元的连线，使其更加整齐美观。

## 优化内容

### 1. 原始设计分析
**原始设计问题**:
1. **中间水池到回用泵**: 三个回用泵的垂直段长度不同（70、150、230像素），不整齐
2. **回用泵到至用水单元**: 连线模式不统一（1#和2#是L形，3#是直接水平）
3. **缺乏共享线段**: 每个连线独立，视觉上不够整齐

### 2. 优化后设计
**优化设计特点**:
1. **共享水平线**: 中间水池到回用泵使用共享水平线Y=540
2. **共享垂直线**: 回用泵到至用水单元使用共享垂直线X=1125
3. **统一模式**: 所有连线使用相同的三段式模式
4. **整齐美观**: 共享线段使连线更加整齐

### 3. 具体优化

#### 3.1 中间水池到回用泵优化
**优化前**:
```
中间水池下点(1225,520) → 各回用泵右边点(1250,Y):
1#: (1225,520) → (1250,520) → (1250,590)  垂直70像素
2#: (1225,520) → (1250,520) → (1250,670)  垂直150像素
3#: (1225,520) → (1250,520) → (1250,750)  垂直230像素
```

**优化后**:
```
中间水池下点(1225,520) → 共享水平线Y=540:
(1225,520) → (1225,540)  垂直20像素

共享水平线 → 各回用泵:
1#: (1225,540) → (1250,540) → (1250,590)  垂直50像素
2#: (1225,540) → (1250,540) → (1250,670)  垂直130像素
3#: (1225,540) → (1250,540) → (1250,750)  垂直210像素
```

#### 3.2 回用泵到至用水单元优化
**优化前**:
```
回用泵左边点(1150,Y) → 至用水单元右边点(1100,750):
1#: (1150,590) → (1100,590) → (1100,750)  L形
2#: (1150,670) → (1100,670) → (1100,750)  L形
3#: (1150,750) → (1100,750)               水平
```

**优化后**:
```
回用泵左边点(1150,Y) → 共享垂直线X=1125:
1#: (1150,590) → (1125,590)  水平25像素
2#: (1150,670) → (1125,670)  水平25像素
3#: (1150,750) → (1125,750)  水平25像素

共享垂直线 → 至用水单元:
1#: (1125,590) → (1125,750) → (1100,750)  垂直160+水平25像素
2#: (1125,670) → (1125,750) → (1100,750)  垂直80+水平25像素
3#: (1125,750) → (1100,750)               水平25像素
```

## 设计原则

### 1. 共享线段设计
- **水平共享**: Y=540，中间水池到回用泵共享20像素垂直线
- **垂直共享**: X=1125，回用泵到至用水单元共享25像素水平线
- **减少重复**: 共享线段避免重复绘制

### 2. 统一连线模式
- **三段式**: 所有连线都是水平-垂直-水平或垂直-水平-垂直
- **正交规范**: 全部水平或垂直连线
- **整齐排列**: 共享线段使连线排列整齐

### 3. 视觉优化
- **对称美观**: 三个回用泵的连线对称排列
- **层次清晰**: 共享线段形成清晰的层次结构
- **专业规范**: 符合工业控制系统连线标准
