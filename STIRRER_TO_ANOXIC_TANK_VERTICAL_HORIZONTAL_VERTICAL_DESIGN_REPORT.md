## 修改内容

### 1. 原始设计分析
**原始设计**（修改前）:
```
路径：搅拌机(730,350) → 垂直向上到(730,320) → 水平向左到(665,320) → 垂直向上到(665,370)
顺序：竖线（向上）→ 横线（向左）→ 竖线（向上）
问题：第一段竖线是向上，不是标准的"先竖线"
```

### 2. 新设计
**新设计**（修改后）:
```
路径：搅拌机(730,350) → 垂直向下到(730,400) → 水平向左到(665,400) → 垂直向上到(665,370)
顺序：竖线（向下）→ 横线（向左）→ 竖线（向上）
特点：标准的"先竖线再横线再竖线"
```

## 设计原则

### 1. 标准三段式设计
- **先竖线**: 从搅拌机垂直向下50像素
- **再横线**: 水平向左65像素
- **再竖线**: 垂直向上30像素直接连接到缺氧池

### 2. 合理流向
- **向下开始**: 从搅拌机向下，符合重力流向
- **水平移动**: 向左移动到缺氧池右侧
- **向上连接**: 向上连接到缺氧池上边

### 3. 直接连接
- **直接连接**: 最后一段竖线直接连接到缺氧池
- **无冗余**: 没有多余的折线或弯角
- **简洁明了**: 三段式设计简洁明了
