## 设计原则

### 1. 连线模式统一
现在搅拌机和加药装置都使用相同的"先下再横再下"连线模式：

**加药装置到缺氧池**:
```
(450,250) → (450,320) → (625,320) → (625,420)
先下70像素 → 横175像素 → 下100像素
```

**搅拌机到缺氧池**:
```
(730,350) → (730,450) → (665,450) → (665,520)
先下100像素 → 横65像素 → 下70像素
```

### 2. 连接点分配
- **加药装置**: 连接到缺氧池顶部左边点`(625, 420)`
- **除臭设备**: 连接到缺氧池顶部中心点`(645, 420)`
- **搅拌机**: 连接到缺氧池底部右边点`(665, 520)`

### 3. 视觉优化
- **路径清晰**: 三段式连线，方向明确
- **避免交叉**: 搅拌机连线在较低位置，避免与其他连线交叉
- **层次分明**: 不同设备连接到不同高度的点
